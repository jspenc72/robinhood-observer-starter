#Example Starter Project to User the Unofficial [Robinhood NodeJS SDK](https://github.com/jspenc72/robinhood-observer)


##Getting started

- Clone this repo

```bash
git clone https://github.com/jspenc72/robinhood-observer-example
```

- Install dependencies
```bash
npm install
```

- Run the Example
```bash
node index.js
```
